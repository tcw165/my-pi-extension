# My Pi Extension

A custom pi coding agent extension with Graphite (gt) CLI skill integration and tools for managing stacked PRs and Git workflows.

## Features

- **Graphite CLI Skill**: Comprehensive documentation for stacked PR management
- **Skills Support**: Google Calendar CLI (gccli) and other integrations
- **Extensible Architecture**: Built on pi's extension framework

## Getting Started

```bash
# Install dependencies
npm install

# Run pi
pi
```

## Development

See `.pi/skills/` for available skills and customizations.
